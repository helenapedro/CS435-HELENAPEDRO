Algorithm countNodesDFS(G):
    for each vertex v in G:
        visited[v] ← false

    count ← 0

    for each vertex v in G:
        if visited[v] = false:
            DFS(v)

    return count


Procedure DFS(u):
    visited[u] ← true
    count ← count + 1

    for each vertex w adjacent to u:
        if visited[w] = false:
            DFS(w)
